# Spitfire.Client

> Fire and forget. No misfires.

.NET client for **Spitfire** — a durable, polyglot background job queue. One binary, no database.

This is a **placeholder release** to reserve the package name. The real client API is under development. See the project at https://github.com/spitfirehq/spitfire.
