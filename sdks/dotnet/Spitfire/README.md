# Spitfire

> Fire and forget. No misfires.

.NET package for **Spitfire** — a durable, polyglot background job queue. One binary, no database.

This is a **placeholder release** to reserve the package name. The real client lives in the `Spitfire.Client` package and is under development. See the project at https://github.com/spitfirehq/spitfire.
